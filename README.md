        <h1>Basketball Counter Points App</h1>
        <p>Welcome to the Basketball Counter Points App! This application allows you to easily keep track of the scores in a basketball game.</p>
        
        <h2>Features</h2>
        <ul>
            <li>Increment and decrement team scores.</li>
            <li>Reset scores to start a new game.</li>
            <li>Simple and intuitive user interface.</li>
        </ul>

        <h2>Installation</h2>
        <p>Follow the steps below to install and run the app:</p>
        <pre>
            <code>
                git clone https://github.com/yourusername/basketball-counter-points-app.git
                cd basketball-counter-points-app
                flutter pub get
                flutter run
            </code>
        </pre>

        <h2>Usage</h2>
        <p>Once the app is running, you can use the following controls:</p>
        <ul>
            <li><strong>Increment:</strong> Click the "+" button to add a point to the team's score.</li>
            <li><strong>Decrement:</strong> Click the "-" button to subtract a point from the team's score.</li>
            <li><strong>Reset:</strong> Click the "Reset" button to reset the scores to zero.</li>
        </ul>
